> Simulador de Venta de entradas

Con este simulador, intento emular una página de venta de entradas para eventos.

Utilizando diferentes funciones de JavaScript, se puede agregar entradas a un carrito de compras y calcular el total.

El código está dividido en varias funciones, cada una encargada de una tarea específica.

Los datos están almacenados en un array de objetos, donde cada objeto representa un evento con su respectivo precio.

El objetivo es practicar la manipulación de arrays y objetos en JavaScript, así como la interacción con el usuario a través de prompts y alerts.

No utilicé console.logs en este proyecto porque prefiero mostrar la información directamente en la página para mejorar la experiencia del usuario.
